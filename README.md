
🌦️ Weather App

A simple and responsive Weather App that allows users to search for real-time weather information of any city around the world. This application fetches live weather data including temperature, humidity, wind speed, and weather conditions using a weather API. Inspired by modern weather dashboard designs and built with clean UI principles.

⚙️ Installation

To run this project locally:

Clone the repository
git clone https://github.com/elamu006/weather-app.git
Navigate to the project folder
cd weather-app
Open index.html in your browser
🔑 API Setup
Get a free API key from:
OpenWeatherMap API
Replace your API key in script.js
const apiKey = "YOUR_API_KEY";
